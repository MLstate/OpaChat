A simple, one-room, scalable real-time web chat, with file sharing - built in Opa.
==================================================================================

1) Requirements
---------------
[Opa](https://opalang.org/get.xmlt) build > 1046

2) Build
--------
$ `make`

3) Launch
---------
$ `./opa_chat.exe`

2+3) Build and run :)
---------------------

$ `make run`

4) Admire
---------

[http://localhost:8080](http://localhost:8080)
